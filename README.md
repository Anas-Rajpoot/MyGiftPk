# MYGIFT Agent Kit

Drop the contents of this folder into your project repo root, then start your coding agent (Claude Code / Sonnet) in that folder.

- CLAUDE.md        -> auto-loaded rules every session
- PHASES.md        -> paste ONE phase prompt at a time
- .claude/skills/  -> 5 pre-built project skills (agent creates more if a gap appears)

Start with: "Read CLAUDE.md and PHASES.md. Begin Phase 0."
